# FIN7 Threat Intelligence Case Study

## Overview

This project analyzes the financially motivated cyber threat group FIN7 and evaluates how its operations align with the phases of a structured cyber operation lifecycle.

The analysis uses publicly available threat intelligence, MITRE ATT&CK mappings, and documented FIN7 tactics, techniques, and procedures (TTPs).

## Topics Covered

- Threat Intelligence
- MITRE ATT&CK
- FIN7
- Phishing Campaigns
- Persistence Techniques
- Ransomware Operations
- Cyber Operation Lifecycle
- Threat Actor Analysis
